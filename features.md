New Feature Requirements
1. Core Modules
1.1 Cryptocurrency
P2P Trading: Robust Peer-to-Peer marketplace for users to trade assets.
Buy & Sell: Direct purchase and sale of cryptocurrencies.
Send & Receive: On-chain and internal transfers.
Portfolio Management: Visual tracking of crypto assets and total value.
1.2 Fiat & Payments
Send & Receive: Bank transfers and internal fiat transfers (USD, NGN).
Bill Payment: Airtime, Utilities, Data bundles.
Giftcards: Buy and sell functionality.
Virtual Cards: Issuance for online spending (USD cards).
1.3 Savings (Piggyvest-style)
Flexible Savings: Deposit/Withdraw at will.
Target Savings: Goal-oriented savings with locked funds/dates.
Interest: Automated interest calculation.
1.4 User Management
Authentication: Secure login, 2FA, Biometric support (Fingerprint/FaceID via mobile).
KYC Identity Verification: Integration with YouVerify.
Profile: Comprehensive user settings and security controls.
History: Detailed transaction logs for all activities.
Security: Pin setup, Biometrics, Password management.
2. Admin & System
Admin Portal: Complete backend for user management, transaction monitoring, and fee configuration.
Multi-Currency: Native support for USD and NGN.
3. Integrations Identified
KYC: YouVerify (Replacing/Enhancing current setup).
Virtual Cards: Existing Strowallet/Payscribe (To be confirmed if keeping).
Bills: Payscribe (To be confirmed).
